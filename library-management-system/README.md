# Library Management System

## 📖 Description
This is a simple Library Management System developed in Python.

## ✨ Features
- Add Book
- View Book
- Search Book
- Delete book
- Exit

## 💻 Language
Python

## ▶️ How to Run
1. Open the project in VS Code.
2. Open Terminal.
3. Run:

```bash
python main.py
```
